# Phase 9: Automated Testing & CI/CD Pipelines Plan

<objective>
Establish a rigorous automated testing culture and CI/CD infrastructure to maintain high software quality and enable rapid, confident releases.
</objective>

<tasks>

### Task 1: Comprehensive Backend Test Suite
- **Pytest Framework**: Set up `pytest` with `pytest-asyncio` for the FastAPI worker.
- **Unit Tests**: Mock external APIs (OpenAI, Supabase) to test scraper logic and enrichment formatting in isolation.
- **Regression Tests**: Add test cases for all "auto-fixed" bugs identified in previous audits.

### Task 2: Frontend & E2E Testing
- **Vitest/Jest**: Implement unit tests for the `usePersonalization` hook and formatting utilities.
- **Playwright E2E**: Script automated flows for:
  - User Login -> Subscription Check -> Run Scrape -> View Leads.
  - Stripe Checkout redirect and success handling.

### Task 3: CI/CD Pipeline (GitHub Actions)
- **Workflow Automation**: Create `.github/workflows/main.yml` to:
  - Lint Python and TypeScript code.
  - Run the full test suite on every push to `master`.
  - Build the Docker image to verify container integrity.
- **Automated Deployments**: Configure CD to deploy the staging environment upon successful test completion.

</tasks>

<success_criteria>
- No code can be merged into the main branch without passing all automated tests.
- Full E2E suite completes in under 5 minutes.
</success_criteria>
